#  Grocery List & Budget Management App

A modern Android application built with **Kotlin** and **Jetpack Compose** that helps users manage grocery shopping while staying within their budget. The app allows users to create grocery lists, track expenses in real time, and synchronize data securely across devices using Firebase.

---

##  Features

*  Secure user authentication with Firebase Authentication
*  Real-time cloud synchronization using Firebase Firestore
*  Create, edit, and delete grocery items
*  Set and manage a shopping budget
*  Automatically calculate remaining budget
*  Offline data storage using Room Database
*  Store user preferences with DataStore
*  Modern UI built with Material Design 3
*  Responsive and efficient state management using Kotlin Coroutines

---

##  Architecture

This project follows the **MVVM (Model-View-ViewModel)** architecture to keep the code clean, scalable, and maintainable.

##  Getting Started

### Prerequisites

* Android Studio (latest stable version)
* JDK 17 or later
* Android SDK
* Firebase Project

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/grocery-budget-manager.git
```

2. Open the project in Android Studio.

3. Connect your Firebase project by adding your `google-services.json` file to the `app/` directory.

4. Sync Gradle.

5. Build and run the application.
